¿Qué es Python?


Python es un lenguaje de programación poderoso y fácil de aprender. Cuenta con estructuras de datos eficientes y de alto nivel y un enfoque simple pero efectivo a la programación orientada a objetos. La elegante sintaxis de Python y su tipado dinámico, junto con su naturaleza interpretada, hacen de éste un lenguaje ideal para scripting y desarrollo rápido de aplicaciones en diversas áreas y sobre la mayoría de las plataformas.

El intérprete de Python y la extensa biblioteca estándar están a libre disposición en forma binaria y de código fuente para las principales plataformas desde el sitio web de Python, http://www.python.org/, y puede distribuirse libremente. El mismo sitio contiene también distribuciones y enlaces de muchos módulos libres de Python de terceros, programas y herramientas, y documentación adicional.

El intérprete de Python puede extenderse fácilmente con nuevas funcionalidades y tipos de datos implementados en C o C++ (u otros lenguajes accesibles desde C). Python también puede usarse como un lenguaje de extensiones para aplicaciones personalizables.

¿Quienes somos?

En Python Venezuela somos un grupo de entusiastas reunidos con el objectivo de promocionar y difundir las bondades del lenguaje de programación Python. Así mismo el intercambio de experiencias hace de nuestro grupo el sitio idóneo para aprender, compartir, impulsar iniciativas y reclutar programadores.

Si tiene curiosidad o alguna inquietud sobre Python no dude en contactarnos, comentarios, sugerencias, dudas e información serán bienvenidas.

Actividades

En PyVE llevamos a cabo diversas actividades para la promoción del uso del lenguaje Python como herramienta para de desarrollo, algunas de estas actividades se describen a continuación

PyCon

Es la conferencia de desarrolladores Python, organizada por los miembros de la comunidad local. Usualmente consta de varios tracks con diversos temas o enfoques técnicos o no técnicos para entusiastas de todos los niveles.

En un PyCon tendrás la oportunidad de:

  • Presenciar una gran variedad de presentaciones, tutoriales y charlas relámpago.
  • Aprender acerca de los avances en la comunidad de desarrolladores Python.
  • Conocer pythonistas de otras partes del mundo.
  • Inscribirte en tutoriales impartidos por expertos.
  • Participar en jornadas de programación (sprints) con otros entusiastas.

Página del PyConVE 2012

PyDay

Un PyDay es una jornada de un día dedicada a Python, puede ser una jornada de iniciación al lenguaje y a algunas de las herramientas basadas en Python.

PyGouts

De vez en cuando, a través de las listas de correo, surge algún interés sobre un tema o herramienta específico. Usualmente alguien conoce y usa la herramienta y ofrece un taller vía Google Hangout para dar una introducción y satisfacer el interés de los miembros de la comunidad.

PyMeeting

Reuniones habituales para planificar eventos, hablar de los proyectos en los que trabajamos, compartir conocimiento o simplemente compartir un tiempo como comunidad, se anuncian a través de la lista de correos.
